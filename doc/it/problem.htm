GeneWeb: problemi


Ecco qualche problema riscontrato di frequente in "gwd". Avete lanciato il daemon GeneWeb, "gwd" che risponde normalmente come indicato nel capitolo "Come iniziare con GeneWeb" al paragrafo "Comando gwd".

Improvvisamente...

Messaggio "socket failed"

Sotto Windows, se questo messaggio appare nella finestra interattiva e non riuscite a connettervi all'indirizzo indicato usando il vostro navigatore (browser) web, forse il protocollo internet (TCP/IP) non è installato sul vostro computer.

In questo caso, installatelo. Si trova in "Pannello di controllo", "cartella rete". In seguito rilanciate "gwd" e provate di nuovo.

Tentativo di collegamento alla rete

  • Quando indicate l'indirizzo nel vostro navigatore (browser) web, ad esempio : 

    http://localhost:2317/pippo
    Il navigatore (browser) web risponde che bisogna connettersi alla rete, che il programma non può funzionare in modo non collegato.

    Questo problema è stato constatato con certe versioni d'Internet Explorer.

  • Abbiamo ricevuto molti messaggi in questo senso da parte degli utilizzatori di GeneWeb (grazie). Ciò può dipendere dal numero di versione utilizzato :

    • In Internet Explorer, selezionate il menu "Strumenti", "Opzioni". Poi, nella finestra "Connessioni", "Impostazioni" attivate "Non utilizzate il server proxy per gli indirizzi locali".

    • [Versione 3.02] In Internet Explorer, selezionate il menu "Strumenti", "Opzioni", "Connessione". In seguito andate a "Server proxy" e selezionate "Installazione", aggiungete in "Eccezioni" e in "Non utilizzare il server proxy per gli indirizzi che cominciano con" gli indirizzi dati all'avvio di gwd.exe. Per esempio (notate i punti e virgola tra i nomi) : 
      127.0.0.1 ;localhost ;computer

    Se il programma si rifiuta ancora di funzionare, vi consigliamo di utilizzare un altro navigatore, come Netscape (distribuito gratuitamente).

Cannot access file "start.txt "

  • Un altro problema frequente è costituito dal messaggio : "Cannot access file start.txt".
  • Se utilizzate GeneWeb sotto Windows, avete forse scompattato l'archivio usando "pkunzip". In questo caso bisogna scompattare usando le opzioni "-e -d" : 
    pkunzip -d -e geneweb.zip
    Se non l'avete scompattato in questo modo, ricominciate con le buone opzioni.
