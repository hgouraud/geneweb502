Pour demarrer avec GeneWeb, il va vous falloir d'abord effectuer
quelques commandes.

Le plus simple est de lancer le programme "gwsetup". Il vous permettra
d'effectuer ces commandes par l'intermediaire de votre navigateur
Web. Tout vous sera explique dans la navigation.

   Sous Windows, double-cliquez sur "gwsetup".
   Sous Unix, lancez "gwsetup" dans un xterm.
   Sous MacOS X, lancez "gwsetup" dans un Terminal.

Mais si vous preferez connaitre les commandes de base de GeneWeb, et
que vous etes un habitue des commandes interactives (MSdos, shell),
allez dans le repertoire "gw" et lisez le fichier LISEZMOI.txt. C'est
une autre facon de demarrer.
