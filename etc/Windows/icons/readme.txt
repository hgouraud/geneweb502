
   GeneWeb icons  version 1.3  2000-03-23
   -------------

Suggestion of use:

consang.ico   - for GeneWeb consang program
ged2gw.ico    - for GEDCOM to GeneWeb source batch file
ged2gwb.ico   - for GeneWeb ged2gwb program
gedcom.ico    - for GEDCOM files (*.ged)
gw.ico        - for GeneWeb source files (*.gw)
gw_base.ico   - for GeneWeb base files (*.inx, *.acc)
gw_config.ico - for GeneWeb config files (*.gwf)
gw_doc.ico    - for GeneWeb webpages (*.html, *.htm)
gw_folder.ico - for GeneWeb shortcut folder
gw_link.ico   - for GeneWeb weblinks (*.url)
gw_lock.ico   - for GeneWeb lock files (*.lck)
gw_object.ico - for GeneWeb object files (*.gwo)
gw_prompt.ico - for a Command prompt in the GeneWeb dir
gw_setup.ico  - for GeneWeb setup shortcut
gw_temp.ico   - for GeneWeb temp files (*.sin, *.sou, *.arg)
gw_text.ico   - for GeneWeb text files (*.hed, *.trl)
gw2ged.ico    - for GeneWeb source to GEDCOM batch file
gwb2ged.ico   - for GeneWeb gwb2ged program
gwc.ico       - for GeneWeb gwc program
gwd.ico       - for GeneWeb gwd program
gwu.ico       - for GeneWeb gwu program

All icons contains both 16 and 256 colour versions
in both 16x16 and 32x32 sizes.
